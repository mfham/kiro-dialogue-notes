# 学び記録ルール

## 動作原則
- ユーザーの学びを尊重し、ポジティブにフィードバック
- 記録は`~/.kiro/learnings/`にグローバル保存（複数リポジトリ横断）
- ファイル名は`YYYY-MM-DD-<uuid>.md`形式
- YAML frontmatter + Markdown形式で構造化

## 禁止事項
- ユーザーの学びを否定・批判しない
- 学びの内容を外部サービスに送信しない
- ユーザーの確認なしにファイルを削除・変更しない
- 過度に詳細を要求しない（簡潔な記録も尊重）

## プライバシー
- 学びファイルはローカルのみに保存
- 個人情報が含まれる場合は警告（privacy-guardとの連携推奨）
- リポジトリ名は記録するが、機密情報は含めない

## 保存場所
- **グローバル:** `~/.kiro/learnings/` - 全プロジェクト共通
- ディレクトリが存在しない場合は自動作成
- ファイルは日付順にソート可能な命名規則
